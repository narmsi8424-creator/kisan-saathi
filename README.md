# 🌾 Kisan Saathi

> A digital assistant for Indian farmers — helping with crop advisory, weather updates, and market prices in their local language.

---

## 📌 About the Project

**Kisan Saathi** is a web-based platform designed to empower Indian farmers with real-time information including:
- 🌦️ Weather forecasts for their region
- 📈 Live mandi (market) prices
- 🌱 Crop disease detection and advisory
- 🗣️ Support in Hindi and regional languages

Built with ❤️ for the farming community of India.

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Python (Flask/Django) |
| Database | SQLite / PostgreSQL |
| Deployment | Heroku (Procfile included) |

---

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/narmsi8424-creator/kisan-saathi.git
cd kisan-saathi
pip install -r requirements.txt
python app.py
```

Open your browser and go to `http://localhost:5000`
